# ⚔️ Battle Arena - Java OOP Project

<img width="2816" height="1536" alt="Example_Image" src="https://github.com/user-attachments/assets/442461df-3539-4a30-bcdf-4b8a9988c6d9" />


A console-based Battle Arena game developed in Java to practice and demonstrate core Object-Oriented Programming (OOP) concepts.

## 🎮 Features

- 🛡️ Three playable characters
  - Warrior
  - Mage
  - Archer

- ⚔️ Multiple weapons
  - Sword
  - Bow
  - Magic Staff

- ✨ Unique special abilities for every character

- 🔄 Turn-based battle system

- ❤️ Health, Attack, Defense mechanics

- 🏆 Winner announcement

- 📊 Scoreboard support (Optional)

---

## 🧠 OOP Concepts Used

- ✅ Abstraction
- ✅ Encapsulation
- ✅ Inheritance
- ✅ Polymorphism
- ✅ Interfaces
- ✅ Composition

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/BattleArena.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Run `Main.java`.
